## Battle-e — Status: Blocked (no .raw dotcode files available)

USA Battle-e cards are **not available as `.raw` dotcode files**. The collection only contains `.sav` GBA save memory dumps (128 KB each) — a different format that cannot be converted to printable dotcodes without reverse-engineering the encoded card data from the save state.

The Japan Battle-e+ cards (08-A001 through 08-A048) exist as `.raw` files but require a **Japan e-Reader+** peripheral. They are not compatible with a USA e-Reader or English Ruby/Sapphire.

**To unblock Battle-e support**, the following would be needed:
- USA `.raw` dotcode files for English Battle-e Series 1 cards, OR
- A converter that can extract dotcode data from the `.sav` memory dumps
